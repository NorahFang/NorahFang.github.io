Git基本命令

2016-10-25 Views git 前端开发 2 min read             繁

创建git仓库 $ git init

下载项目的最新更改
该命令用于拉取某分支的最新副本（建议工作时每次都输入这个命令）。
origin（默认远程库的名称）
git pull origin 远端 分支名称 -u

创建一个分支
git checkout -b 分支名称

在某分支上进行开发
git checkout 分支名称

将更改加入到本次提交
当输入"git status"时，您的更改会显示为红色。
git add 红色的修改

浏览您所做的更改
git status

正式提交更改
git commit -m "提交的描述"

查看修改内容
git diff

提交您的更改到服务器
git push origin 分支名称

删除代码库的所有更改（不包含提交到暂存区的变更）
git checkout .

删除代码库的所有更改（包含未跟踪的文件）
git clean -f

将某分支合并到master分支
git checkout 分支名称
git merge master

查看执行过的命令
history

删除一个文件
rm 文件名

删除一个目录及其中的所有内容
rm -rf 目录名

查看提交日志
git log

提取远程仓库
从远程仓库下载新分支与数据：
git fetch origin

从远端仓库提取数据并尝试合并到当前分支
git pull origin 分支
