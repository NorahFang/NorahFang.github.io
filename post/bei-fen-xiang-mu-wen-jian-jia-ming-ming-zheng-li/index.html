备份项目文件夹命名整理

2018-11-01 Views 常用规范 2 min read             繁

设计项目文件夹

  • docs ：放置项目需求文档等文档材料【如果是网站改版，则需要将原网站的截图2~4张存入此文件夹】
  • design ：视觉源文件
  • output ：截图、展示用的完整视觉稿
  • inspiration：灵感 提供灵感的素材和文档，比如情绪板、截图、UI模式等等
  • assets ：项目相关的设计素材
  • archive ：存放旧版或不再需要的文件
一般我都用.bat文件每次右键运行就自动生成文件夹。

具体操作：

新建txt文档，输入内容

docs
design
output
inspiration
assets
archive

保存并命名为 【设计文件夹列表.txt】放在E盘

再新建txt文档，在文档内输入

@echo 请先将要新建文件夹的名称保存到文件夹名列表.txt（一个文件夹为一行，子文件"\"连接）
@pause
@for /f %%a in (E:\设计文件夹列表.txt) do (md %%a)
@echo 批量新建文件夹完成
@pause

保存并修改后缀名为.bat的文件。

批量生成设计文件夹.bat

↑已经设置好的 文件

psd命名：

页面层级+页面内容+子层层级+页面内容+保存时间+v版本号

用下划线连接，不推荐使用驼峰（识别性不高）

如果某个psd在当天改过多次，日期后可加上版本号

例如：会员中心>我的订单>【查看订单】页面

1_会员中心_3_查看订单_20170203.psd
